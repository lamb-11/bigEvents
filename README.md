# Vue2大事件

>  基于Vue2+ElementUI组件库完成的后台文章数据管理项目



## 里面所涉及到的知识点：

1. 组件封装
2. vue-router路由
3. vuex全局共享数据
4. axios：baseURL、请求拦截器、响应拦截器
5. elementUI组件库的使用
6. ES6模块化语法
7. 富文本编辑器的简单使用
8. 熟悉ESLint
9. git的基本使用
10. 项目结束打包
